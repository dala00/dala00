# CLAUDE.md

このリポジトリは GitHub プロフィール用（`dala00/dala00`）。`README.md` がプロフィールトップに表示される唯一の成果物。

## 編集ルール

- 構成は「自己紹介 → カウンター/キリ番リンク → Contents → 各セクション → 連絡先」。この順序と見出し（`##`）構造を保つ。
- 作品を追加するときは、該当セクションに1ブロック（`### [名前](URL)` ＋ 1〜3文の説明）を追記する。新セクションを足したら `Contents` の目次も更新する。
- 各作品の説明は「機能の列挙」より「何のための物か」を一文で言い切る。
- 曲・動画は中身がテキストとして読めないため、曲MVには必ず「どんな曲か」の説明文を添える。
- 冒頭のカウンター画像とキリ番報告リンクは消さない。
