## つくったサービス

### [AIバトラー（AI Battler）](https://ai-battler.com)
ユーザーが作ったキャラクター同士を **AI が判定して戦わせる、テキストベースの対戦ゲーム**。
プレイヤーはステータス・必殺技・プロンプトでキャラクターを設計し、相手に挑みます。AI が勝敗と「判定文（バトルの描写）」を生成して返すのが特徴です。
1on1 バトル / 複数人で集まるグループバトル / ステータス連動で遊ぶワールド（RPG 的コンテンツ）の3モードがあり、Flutter アプリと Web の両方から遊べます。

📱 [Google Play（Android）](https://play.google.com/store/apps/details?id=com.alphabrend.aibattle&hl=ja) ／ [App Store（iOS）](https://apps.apple.com/jp/app/ai-battler-ai-judge-battle/id6448725330)

## Claude Code プラグイン

### [claude-marketplace](https://github.com/dala00/claude-marketplace)
個人開発者向けの **Claude Code プラグインマーケットプレイス**。
- **security-check** … プロジェクト構成を自動判定し、Web フロント（XSS・CORS 等）/ モバイル・デスクトップアプリ / バックエンド（SQLインジェクション対策等）/ クライアント側DB（Firebase 等）/ 法務・コンプライアンスのセキュリティ検査を実行するツール。
- **business-idea-check** … ビジネスアイデアを「ターゲット層」「収益性」「集客」「開発可能性」など6観点から5段階評価し、成功可能性を判定するツール。
  - 自分で試した判定結果のストック → [Zenn スクラップ](https://zenn.dev/dala/scraps/b9a4765828dee9)

### [claude-game-marketplace](https://github.com/dala00/claude-game-marketplace)
**Claude Code 上で対話しながらゲームを遊べるマーケットプレイス**。
各プラグインが1本のゲームに対応し、スキル/スラッシュコマンドとして機能します。推理アドベンチャー（『ふくろう堂のラストオーダー』『暁の聖杯』『停電チャンネル』『走馬灯の雪』など）やリバーシなどを収録。`/new-adventure` で自作ゲームも作れます。

## オリジナル曲 / MV

YouTube チャンネル「**だらだら音楽スタジオ**」（[@daladalamusic](https://www.youtube.com/@daladalamusic)）で公開しているオリジナル楽曲の MV です。
昔つくった弾き語りやテキストサイト時代の自作BGMを、AI・打ち込みで現代に蘇らせたノスタルジックな作品が中心です。
▶ [プレイリスト「オリジナル曲」](https://www.youtube.com/playlist?list=PLzux1lOzhh7iO_wePhNufXLH8YuYKXjSk)

### [ブラウン管](https://www.youtube.com/watch?v=nB_daaTawfk)
手書き風モノクロアニメーションが特徴のMV。昔つくったオリジナル曲（弾き語り音源）を AI で音源化し、深夜の静けさを思わせる少しダークで幻想的な世界観に仕上げた一曲。

### [ぜひまた見に来て](https://www.youtube.com/watch?v=iXvrViC8u1M)
個人開発をテーマにしたポップで切ないアニメ風MV。「全然誰も来ない」寂しさを抱えながらも「なぜかまた作ってしまう」創作への情熱と、「ぜひまた見に来て」という願いを歌う。

### [ロールプレイ（Always in the dark）](https://www.youtube.com/watch?v=-mbuzIZ5xjw)
ダーク路線のRPGをモチーフにしたオリジナル曲のMV。黒き力からバフ（能力強化）を受けるようなシリアスな戦闘展開を想起させる、重厚な世界観の一曲。

### [shooting](https://www.youtube.com/watch?v=bVG3hYMFiO4)
シューティングゲームのスピード感・浮遊感をイメージした、インスト／弾き語りベースの楽曲。ゲームミュージックのような展開と、レトロで味のあるメロディラインが特徴。

### [diary](https://www.youtube.com/watch?v=3U51xLeDLvQ)
歌詞のないインストゥルメンタル。かつてテキストサイトのトップページで BGM に流していた自作曲を打ち込みで現代に復活させた、日記をつづるように静かでエモーショナルな一曲。
